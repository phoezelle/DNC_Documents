OpenSprinkler v2.0 (2nd generation)
Changes from 1st generation:

- Microcontroller upgraded to ATmega644
- Switching regulator upgraded to LM2596S-5.0 (from MC34063)
- Added microSD card slot
- Added support for LCD backlight and contrast adjustments
- Added pull-up resistor (RPR) for automatic detection of extension boards.
- More digital and analog I/O pins, and extra pin headers for connecting RF transmitter.

Schematics and Board design files require EagleCAD v6.0 or above.
