OpenSprinkler v2.1 (2nd generation)
Changes from 2.0

- Removed ATtiny45 (used to serve as USBtinyISP programmer).
- ATmega644 now running at 12MHz and preloaded with USBasp bootloader.
- Added per-station TVS (smt version) / MOV (diy version).
- Added 120V/2A mini-relay (connected to pin D14.
- Added 2A fuse on the 24V AC line.
- 24V AC terminal block changed to orange colored 3.96mm spacing.

Schematics and Board design files require EagleCAD v6.0 or above.
