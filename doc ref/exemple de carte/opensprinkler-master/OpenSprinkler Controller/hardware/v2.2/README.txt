OpenSprinkler v2.2 (2nd generation)
Changes from 2.1

- Added CH340G USB-serial converter
- ATmega644 now running at 16MHz and preloaded with optiboot bootloader

DIY v2.2_rev and SMT v2.2

- Changed to use MC34063 switching regulator (previously LM2596)
- Added Microchip 24AA02E48 hardware MAC chip

Schematics and Board design files require EagleCAD v6.0 or above.
