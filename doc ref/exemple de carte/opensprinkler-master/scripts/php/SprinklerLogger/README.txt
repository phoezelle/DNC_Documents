This folder contains PHP scripts for logging OpenSprinkler using a cron job. The scripts periodically (every 20 seconds) polls the status of the OpenSprinkler and logs the changes of each poll. 

The scripts are written by David B. Gustavson. For details, please check the Google forum post:

https://groups.google.com/d/topic/rayshobby/dVFQO83AdO8/discussion
